# Week 1 - Office Space Rental App

## Objective

To understand and implement React JSX by creating elements, attributes, objects, lists, and applying conditional CSS styling.

## Features

- Display the heading using JSX.
- Display office space images.
- Store office details using JavaScript objects.
- Display multiple office spaces using the `map()` function.
- Apply conditional styling:
  - Rent ≤ ₹50,000 → Red
  - Rent > ₹50,000 → Green

## Technologies Used

- React JS
- JavaScript (ES6)
- JSX
- HTML5
- CSS3

## Concepts Covered

- JSX Elements
- JSX Attributes
- JavaScript Objects
- Rendering Lists using `map()`
- Conditional CSS
- React Components
