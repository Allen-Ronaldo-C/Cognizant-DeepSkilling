# Week 3 - Blogger App

## Objective

To understand and implement conditional rendering in React by displaying Book Details, Blog Details, and Course Details using different rendering techniques.

## Features

- Display Course Details.
- Display Book Details.
- Display Blog Details.
- Implement multiple approaches to conditional rendering.
- Render lists using the `map()` function.

## Technologies Used

- React JS
- JavaScript (ES6)
- HTML5
- CSS3

## React Concepts Covered

- Functional Components
- Props
- Conditional Rendering
- Logical AND (`&&`)
- Ternary Operator (`?:`)
- Rendering Lists using `map()`
