# Week 2 - Event Examples App

## Objective

To understand and implement event handling in React using buttons, synthetic events, event handlers, state management, and form handling.

## Features

- Increment counter using React state.
- Decrement counter.
- Invoke multiple methods using a single button click.
- Display welcome message using parameterized event handling.
- Handle synthetic click events.
- Convert Indian Rupees to Euro using a form submission event.

## Technologies Used

- React JS
- JavaScript (ES6)
- HTML5
- CSS3

## React Concepts Covered

- Class Components
- State
- Event Handling
- Synthetic Events
- onClick
- onChange
- onSubmit
- Forms
- Multiple Event Methods
