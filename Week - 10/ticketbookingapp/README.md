# Week 3 - Ticket Booking App

## Objective

To understand and implement conditional rendering in React by displaying different pages for guest users and logged-in users.

## Features

- Display Guest page by default.
- Show **Login** button for guest users.
- Show **Welcome back** page after login.
- Show **Logout** button for logged-in users.
- Switch between guest and user pages using React state.

## Technologies Used

- React JS
- JavaScript (ES6)
- HTML5
- CSS3

## React Concepts Covered

- Conditional Rendering
- Functional Components
- Class Components
- State Management
- Props
- Event Handling
