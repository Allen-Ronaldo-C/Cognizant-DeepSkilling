# Week 3 - Ticket Raising App

## Objective

To understand React Forms by implementing a complaint registration system using controlled components, event handling, and state management.

## Features

- Register employee complaints.
- Enter employee name.
- Enter complaint details.
- Generate a random transaction/reference ID.
- Display a confirmation alert after successful submission.
- Reset the form after submission.

## Technologies Used

- React JS
- JavaScript (ES6)
- HTML5
- CSS3

## React Concepts Covered

- Class Components
- State Management
- Controlled Components
- Forms
- Event Handling
- handleChange()
- handleSubmit()
