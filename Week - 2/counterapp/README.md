# Week 1 - Counter App

## Objective

To understand and implement state management in React class components using constructors and the `setState()` method.

## Features

- Login button to count people entering the mall.
- Exit button to count people leaving the mall.
- Uses React state for updating counts dynamically.
- Demonstrates event handling using class components.

## Technologies Used

- React JS
- JavaScript (ES6)
- HTML5
- CSS3
