# Week 1 - Shopping App

## Objective

To understand and implement React components with **props** and **lists** by displaying shopping cart items dynamically in a tabular format.

## Features

- Displays a list of ordered shopping items.
- Uses React class components.
- Passes data using **props**.
- Renders data dynamically using the **map()** function.
- Displays item name and price in a table.

## Technologies Used

- React JS
- JavaScript (ES6)
- HTML5
- CSS3
